# BazaarExpress Local Variables & Links

## Frontend Application
To start the frontend application, open a terminal in the root `Vendor Verse` folder and run a local development server, for example:
```bash
npx -y http-server -p 3000
```
or 
```bash
python -m http.server 3000
```

**Local Host Link (Frontend):** 
[http://localhost:3000](http://localhost:3000)

---

## Backend API Server
To start the backend server, make sure MongoDB is running locally on port 27017. Then, open a terminal in the `Vendor Verse/backend` folder and run:
```bash
npm start
```
*(Runs on port 5000 by default)*

**Local Host Link (Backend):** 
[http://localhost:5000](http://localhost:5000)
[http://localhost:5000/api/auth](http://localhost:5000/api/auth)
